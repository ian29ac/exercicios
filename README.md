# Exercícios de Programação

Repositório com resoluções de problemas e exercícios de algoritmos desenvolvidos em linguagem C, com foco em lógica de programação, estruturas de dados e resolução de problemas computacionais.

## Plataformas

- Beecrowd
- LeetCode
- Codeforces
- AtCoder

## Conteúdos praticados

- Estruturas condicionais e repetição
- Funções e modularização
- Structs e ponteiros
- Alocação dinâmica
- Pilhas, filas e listas encadeadas
- Algoritmos de busca e grafos (BFS / DFS)

## Objetivo

Consolidar fundamentos de programação e desenvolver raciocínio lógico voltado para Engenharia de Computação e desenvolvimento de software.
